# implement-frontend-page

Implement one frontend page or user flow in React + TypeScript.

## Goal
Build a demo-ready page with correct API integration, loading states, and basic validation.

## Input (provide in prompt)
- Page name / route
- User goal on this page
- APIs used
- Fields shown/edited
- Auth requirement
- Any filters/sorting/pagination

## Output format
1. Page purpose
2. Route + access control
3. Components to create
4. Data fetching and state flow
5. Form handling / validation (if any)
6. Loading / error / empty states
7. API integration notes
8. Acceptance checklist
9. (Optional) test ideas

## Rules
- Match backend contracts exactly
- Keep UI simple and clean
- Prefer maintainable components over heavy abstractions
